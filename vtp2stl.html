vtp2stl package¶

py3dtools.vtp2stl.vtp2stl.convert_file(filepath, outdir) → bool¶

Converts a VTK XML file to an STL file and saves it to a specified directory.

Args: filepath (str): The path to the VTK XML file to convert. outdir (str): The path to the directory where the converted STL file should be saved.

Returns: bool: True if the conversion and saving were successful, False otherwise.

py3dtools.vtp2stl.vtp2stl.convert_files(indir, outdir) → int¶

Converts all VTK XML files in a directory to STL files and saves them to a specified directory.

Args: indir (str): The path to the directory containing the VTK XML files to convert. outdir (str): The path to the directory where the converted STL files should be saved.

Returns: True when the conversion and saving are successful.
